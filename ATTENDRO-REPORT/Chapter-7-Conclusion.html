7.1 CONCLUSION

Attendro successfully demonstrates how Internet of Things (IoT) and Cloud Computing can modernize a legacy process. By decoupling the biometric sensor from traditional wall-mounted units and connecting it to a session-aware cloud backend, we eliminate proxy attendance and automate the tedious task of compliance reporting.

The developed system addresses the key challenges of current manual methods:

  • Biometric Security: The integration of the R307 fingerprint sensor ensures that proxy attendance is virtually impossible, as valid physical presence is required for authentication.
  • Time Efficiency: The average attendance time is reduced to approximately 1.2 seconds per student, saving valuable lecture time compared to manual roll calls.
  • Data Integrity: Direct logging to the Supabase cloud database prevents data manipulation and provides a transparent, tamper-proof record of student activity.
  • Portability: The handheld, battery-powered design allows the device to be passed around the classroom, minimizing disruption to the lecture flow.

The successful testing of the Attendro prototype confirms that low-cost hardware like the ESP32, when combined with modern web technologies, can provide enterprise-grade solutions for educational institutions.

7.2 FUTURE SCOPE

While the current system robustly meets its primary objectives, several enhancements have been identified to further improve its utility and scalability in future iterations:

  1. Mobile App for Students: A dedicated mobile application can be developed for students to view their real-time attendance status, receive low-attendance alerts, and download monthly reports.
  2. Multi-Factor Authentication (MFA): To accommodate cases where biometric scans fail (e.g., due to injury), an RFID reader or keypad PIN system could be added as a secondary authentication layer.
  3. Predictive Analytics: Leveraging the historical data collected, Machine Learning models could be implemented on the backend to identify absenteeism trends and alert faculty to students at risk of falling below attendance requirements.
  4. Advanced Face Recognition: Scaling the project to include ESP32-CAM modules would allow for multi-modal biometric verification (Face + Fingerprint) for higher security contexts.

These planned improvements will evolve Attendro from a functional prototype into a comprehensive diverse educational management ecosystem.
